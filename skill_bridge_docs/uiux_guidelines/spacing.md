# 📏 Spacing System

## Spacing Scale (Tailwind)

- 0: `0px`
- 0.5: `2px`
- 1: `4px`
- 1.5: `6px`
- 2: `8px`
- 2.5: `10px`
- 3: `12px`
- 3.5: `14px`
- 4: `16px`
- 5: `20px`
- 6: `24px`
- 7: `28px`
- 8: `32px`
- 9: `36px`
- 10: `40px`
- 12: `48px`
- 16: `64px`
- 20: `80px`
- 24: `96px`
- 32: `128px`

## Usage Patterns

- Use multiples of 4px for all margin/padding
- Section padding: `p-6`, `p-8`, `p-12`
- Card padding: `p-4`, `p-6`
- Grid gap: `gap-4`, `gap-6`, `gap-8`
- Button padding: `px-6 py-3`, `px-8 py-4`

## Do’s

- Use only Tailwind spacing utilities
- Maintain consistent vertical rhythm

## Don’ts

- Don’t use arbitrary px values
- Don’t override spacing in custom CSS
